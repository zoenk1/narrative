Project Statement

This website was a project for a multimedia authoring class project where we were challenged to

create a narrative story in which I decided to utilize the program Skrollr.

My inspiration for this project was drawn from the iconic Super Mario games.

The biggest challenge I faced was creating visually stunning graphics and animations

that were both engaging and coherent with the Super Mario theme. Through diligent research and iteration,

I was able to overcome these challenges and create a website that effectively captured the essence of

Super Mario while showcasing mu technical skills with Skrollr and multimedia authoring.
